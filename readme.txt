Custom Admin Bar allows you to provide access to anything on the web from a convenient drop-down menu in the admin bar!

== Description ==

Easily add your own link or logo to the WordPress Admin Bar.  Then take it a step further by highlighting important links for your users through a simple drop-down menu!

The drag-and-drop ordering makes it easy to create a unique listing of important tasks on your site.

We wanted to make this quick and easy for anyone to setup and use.  We even use it on our popular <a href='http://edublogs.org'>edublogs</a> service to over 1 million bloggers! 

= Custom Admin Bar Features: =
* Select a title or image (logo) for your main image
* Create as many drop-down links as you like
* Easily re-order the drop-down links
* Link to anything: admin area, public side, even another site!
* Pretty-permalink awareness.  So you only need to provide the slug of your page or post.

= Brand your network with the links you want! =
Custom Admin Bar gives you full control over the all-important top-left portion of your site's screen real estate.

== Installation ==
= To Install: =

1.  Download the Custom Admin Bar plugin file
2.  Unzip the file into a folder on your hard drive
3.  Upload the `/custom-admin-bar/` folder to the `/wp-content/plugins/` folder on your site
4.  Visit Network Admin -> Plugins and Network Activate it there.

== Frequently Asked Questions ==

= Do I need to be a paid WPMU DEV member? =
No.  This plugins is offered as is at no charge.  A full-featured, premium version is available for purchase on WPMU DEV.

= Is this BuddyPress compatible? =
The Custom Admin Bar will work with any BuddyPress theme which support the WordPress admin bar.  Any theme compatible from BuddyPress 1.5+ should be fine.

= What size image should I use for the admin bar? =
We recommend 62 X 24, but as long as the height is under 28px you should be fine.
